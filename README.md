# 🤖 AI Job Market Analysis 2025

## 📌 Project Overview
End-to-end data analytics project on Global AI Job Market Trends 2025
using Python, SQL & Power BI — analyzing 15,000+ job listings across 50+ countries.

## 🛠️ Tools Used
- **Python** — Data Cleaning, EDA & Visualizations
- **MySQL** — Database Creation, Table Design & Querying
- **Power BI** — Interactive 4-Page Dashboard

## 📊 Dataset
- **Source:** Kaggle (Global AI Job Market Trend 2025)
- **Link:** https://www.kaggle.com/datasets/pratyushpuri/global-ai-job-market-trend-2025
- **Records:** 15,000 job listings
- **Period:** 2025
- **Region:** 50+ Countries

## 💡 SQL Analysis Performed
- Database & Table Creation
- Total Job Count
- Top 10 Highest Paying Job Titles
- Salary by Experience Level (with Max Salary)
- Top 10 Countries by Job Count & Avg Salary
- Remote vs Hybrid vs Onsite Salary Comparison
- Company Size vs Avg Salary

## 🐍 Python Analysis Performed
- Library Setup (Pandas, NumPy, Matplotlib, Seaborn)
- Dataset Loading & First Look
- Dataset Info & Basic Statistics
- Missing Values & Duplicate Check → Data Cleaning
- Salary Distribution — Histogram
- Top 10 AI Job Titles — Bar Chart
- Salary by Experience Level — Box Plot
- Remote vs Hybrid vs Onsite — Pie Chart
- Correlation Heatmap (NumPy + Seaborn)
- Top 15 Countries by Avg Salary — Bar Chart
- Top 15 In-Demand Skills — Bar Chart
- Average Salary by Industry — Bar Chart
- Company Size vs Avg Salary — Bar Chart

## 🔍 Key Insights
- 🌍 **Total Jobs Analyzed:** 15,000+
- 💰 **Avg Salary:** $115K
- 🏆 **Top Paying Country:** Switzerland ($171K)
- 👔 **Top Job Title:** AI Specialist ($121K)
- 📈 **Salary Correlation:** Experience = 0.74 (strongest driver)
- 🏢 **Large Companies Pay:** 27% more than Small ($130K vs $102K)
- 🌐 **Work Type:** Remote, Hybrid & Onsite equally distributed (~33%)
